CALORIE TRAX


Calorie Trax is a full-stack app that allows users to look up nutritional informaton for over 1,000 common food and beverage items. The data comes from the U.S. Department of Agriculture's MyPyramid Food Raw Dataset. This dataset is available as a CSV file, which I converted to JSON and then imported into MongoDB. The app is built using React, Express and Node. To improve the lookup process for users, I added an autocomplete feature to the search functionality. As the user types in their search term, suggestions that exist in the database appear in a dropdown box and can be selected.

TRIVIA QUIZ


This simple trivia quiz was my first venture into TypeScript and one of my earlier projects using React, so it was an excellent learning experience. The project makes use of of the Open Trivia DB API to serve questions from a wide variety of categories. In addition to allowing the user to select a particular category and difficulty level, I also provided a "Need Help" option. This option removes two incorrect answers from the choices, allowing the user to select their answer from the two remaining options.
